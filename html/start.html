Using HTML to code your website



What is HTML?

HTML is short for Hyper Text Markup Language. It is a language used by almost every website. HTML file usually end in the .html extension.

  • - Codecademy; A interesting webpage, with a lot of programming language lessons, including HTML.  Their HTML Guide
