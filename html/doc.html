Making a basic document



Tags

Every HTML document is made of tags. Tags start with a < and end with a >. Tags usually come in pairs, a opening tag (for example <p>) and a closing tag (for example </p>). Some tags are all alone, they don't have a closing tag (for example <br>)


The DOCTYPE Tag

All HTML documents should begin with a <!DOCTYPE> tag. If you use it correctly, this tag will basically tell your browser (for example Chrome, Internet Explorer, etc.) what HTML version this webpage is in. To use the latest version of HTML (HTML5), use <!DOCTYPE html>.
